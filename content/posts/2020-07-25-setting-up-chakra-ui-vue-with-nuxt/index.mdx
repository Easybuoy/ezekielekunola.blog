To be able to follow up with this article, you must have the following setup on your local environment

- [Node]() installed.

## **What is Nuxt** <a name="what-is-nuxt"></a>

Nuxt.js is a free and open-source web application framework based on Vue.js, Node.js, Webpack, and Babel.js. Find out more [here](https://nuxtjs.org).

## **What is Chakra UI Vue** <a name="what-is-chakra-ui"></a>

Chakra UI Vue gives you a set of accessible and composable Vue components that you can use to build your favourite applications and sites. Find out more [here](https://vue.chakra-ui.com).

## **Setup Nuxt App** <a name="setup-nuxt-app"></a>

There are two ways of setting up a Nuxt application, either by [starting from scratch]() or using the [create-nuxt-app]().

### **Using Create-Nuxt-App** <a name="using-create-nuxt-app"></a>

The create-nuxt-app is a quick and easy way to get started using nuxt. It creates a boilerplate app that is customizable to the user needs.

You can install by using `npx` or `yarn`

> Note: npx comes with ...

For npx, run the command below

```
npx create-nuxt-app <project-name>
```

Where `<project-name>` is the name of the project you want to create.

For yarn, run the command below

```
yarn create nuxt-app <project-name>
```

The above command would prompt you with some questions, and configurations you want for the project. Once you've followed through with the prompts, and the installation is finished, navigate to the project directory on your code editor, open a terminal and run `npm run dev`, or `yarn dev` to start the application.

See example in the image below

And that's it, you've been able to create a Nuxt application, bootstrapped with Chakra UI Vue, you can go ahead to build out your application using Chakra UI Vue. Find out more about the create-nuxt-app setup [here](https://github.com/nuxt/create-nuxt-app)

### **Starting from Scratch** <a name="starting-from-scratch"></a>

If you have an already existing nuxt project, you can skip to [Setting Up Chakra UI](), but if you want to set up a nuxt project from scratch, you can follow the next steps below

- **Setting up Nuxt** <a name="starting-from-scratch"></a>

  - Create project folder and open in your code editor

- INitialize package.json
- install dependencies

```
npm install --save nuxt
```

- Add scripts to package.json

```json
 "scripts": {
    "dev": "nuxt"
  }
```

- Create a pages directory, in the root of the project, and create a index.vue file there.

Add the code below to the index.vue file

```
<template>
  <p>Hello</p>
</template>
```

To run the app, run `yarn dev` on your terminal.

- **Setting up Chakra UI** <a name="starting-from-scratch"></a>
  Install chakra dependencies

```
yarn add @chakra-ui/nuxt @nuxtjs/emotion
```

- Create `nuxt.config` file in the root of the project and add the configuration below

```js
export default {
  mode: "universal",

  head: {
    title: "Chakra UI Vue Starter",
    meta: [
      { charset: "utf-8" },
      { name: "viewport", content: "width=device-width, initial-scale=1" },
      {
        hid: "description",
        name: "description",
        content: "Chakra UI Vue integration with Nuxt starter",
      },
    ],
    link: [{ rel: "icon", type: "image/x-icon", href: "/favicon.ico" }],
  },

  modules: [
    "@chakra-ui/nuxt", // register the chakra nuxt package
    "@nuxtjs/emotion", // register the emotion package which is being used by chakra-ui-vue
    // Doc: https://http.nuxtjs.org
  ],
};
```
